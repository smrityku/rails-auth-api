# Rails Authentication API

## Description
REST API in Ruby on Rails demonstrating user authentication with JWT and OTP.

## Features
- User signup & login
- JWT authentication
- OTP verification (mock)
- Role-based access (user/admin)

## Tech Stack
- Ruby on Rails
- PostgreSQL
- JWT
- Bcrypt

## API Endpoints
- POST /signup
- POST /login

## How to Run
- bundle install
- rails db:create db:migrate
- rails s
